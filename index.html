About this site

This website is a single page of HTML with Tailwind for styling. It doesn't use Webpack or any other build tools. It was made with minor help from GitHub Copilot, though it wasn't very useful.

I chose a vanilla HTML implementation over demonstrating my knowledge of React because it made more sense for the goals of the site: I wanted it to be simple, accessible, and performant.

The source is available on my GitHub if you'd like to see how I made it.
